## Prerequisites
   - Node.js
   - Git Bash
   - XAMPP
   - Expo Go on Android
   - MySQL

## Get started

1. Install dependencies

   ### For app
   ```bash
   cd BarangayEmergency
   npm install expo
   npx expo start
   ```
   Scan QR code in terminal with Expo Go on Android

   ### For database
   ```bash
   cd BarangayEmergency/db
   node index.js
   ```
   ### For website
   ```bash
   cd BarangayEmergency/website/web-app
   npm run dev
   ```
